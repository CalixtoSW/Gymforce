## O que foi feito

Refatorada a Home com UI de gamificação baseada nos endpoints de `/gamification` usando TanStack Query (tier com progresso, stats, histórico de pontos), implementada tela de leaderboard com top 3 em destaque + lista 4-10 + card fixo da posição do usuário, e aplicado refresh do usuário no `authStore` para manter pontos/streak sempre atualizados.

### Decisões tomadas durante a execução

1. Home passou a buscar dados por queries separadas (`summary`, `history`, `qr`, `sheets`) com `staleTime` de 30s para reduzir chamadas redundantes e manter atualização frequente.
2. Leaderboard usa fallback visual de avatar por iniciais quando `avatar_url` é nulo e mantém card fixo inferior com cálculo de pontos faltantes para o top 10.

## Estado atual

### O que funciona ✅
- Home mostra tier badge com progress bar para próximo threshold.
- Home mostra stat cards (streak, pontos, rank), QR code e histórico de 5 eventos.
- Home revalida dados ao focar aba e no pull-to-refresh.
- Leaderboard carrega top 10, destaque para top 3 e lista de posições seguintes.
- Card fixo mostra `my_rank` e pontos faltantes para #10 quando aplicável.
- `refreshUser` é chamado no retorno do app ao foreground e após conclusão de treino.
