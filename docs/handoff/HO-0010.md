### Decisões tomadas durante a execução

1. `GamificationService.credit_points/debit_points` recebeu parâmetro `commit` para suportar transações atômicas no fluxo de resgate/cancelamento de recompensas.
2. Respostas de `RedemptionResponse` retornam relacionamento `reward` já carregado via `selectinload` para evitar lazy-load assíncrono durante serialização.
